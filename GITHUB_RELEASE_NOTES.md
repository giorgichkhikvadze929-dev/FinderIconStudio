# Finder Icon Studio v0.1.0

Early beta release of Finder Icon Studio for macOS.

## Install

1. Download `Finder Icon Studio.zip`.
2. Unzip it.
3. Move `Finder Icon Studio.app` to Applications.
4. Right-click the app and choose Open.

If macOS says the app is damaged, run:

```sh
xattr -dr com.apple.quarantine "/Applications/Finder Icon Studio.app"
```

Then open it again.

## Notes

This app is unsigned and not notarized yet, so macOS may show security warnings.
